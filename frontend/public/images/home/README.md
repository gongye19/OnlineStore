# 主页图片说明

请将以下三张图片放在此目录下：

1. **hero.jpg** (或 hero.png) - Hero 背景图
   - 建议尺寸：2000px 宽，横向
   - 用途：主页顶部大背景图

2. **philosophy.jpg** (或 philosophy.png) - 工艺哲学配图
   - 建议尺寸：1200px 宽，竖向（4:5 比例）
   - 用途：工艺哲学部分右侧展示图

3. **founder.jpg** (或 founder.png) - 主理人照片
   - 建议尺寸：800px 宽，竖向（3:4 比例）
   - 用途：主理人介绍部分

## 图片格式支持
- JPG / JPEG
- PNG
- WebP

## 注意事项
- 图片文件名必须与配置中的名称一致（hero, philosophy, founder）
- 图片会自动从 `/images/home/` 路径加载
- 如果图片不存在，会显示加载失败提示
